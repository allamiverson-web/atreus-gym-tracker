# Atreus Gym Tracker

A dark cyberpunk workout tracker draft built as a static GitHub Pages-ready website.

## What works

- Start, pause, and finish a gym session timer.
- Tap the character muscles or muscle shortcut buttons to pick a workout area.
- Log exercise, sets, reps, and weight.
- Save progress in the browser with `localStorage`.
- View workout days, 7-day rep trends, and muscle XP balance.

## Preview locally

Open `index.html` directly, or serve this folder with any static web server.

## Publish on GitHub Pages

Put these files in a GitHub repository, then enable Pages from the repository settings using the main branch root.
